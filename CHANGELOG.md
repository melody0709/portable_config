# Changelog

## V1.6.4 - 2026-03-24

- 合并 `path` 双监听为单监听，修复 M3U 初始加载与频道历史跟踪的顺序冲突，减少首台闪切
- 回看续播改为纯 `end-file` 事件驱动，移除轮询定时器与续播后 `seek` 流程
- 回看续播时间链路调整为：`next_start = start_utc + duration`，`next_end = next_start + 5h`
- 新增回看时长缓存：解决 `end-file` 时 `duration=nil` 导致无法推算下一段起点的问题
- 增强回看 URL 查询参数覆盖：统一重写 `starttime/endtime` 及常见运营商字段（含 `ztestarttime/zteendtime`）
- 新增回看续播调试日志，便于定位 `next_start/new_url` 生成过程

## V1.6.3 - 2026-03-23

- 修复菜单横向抖动：分组/日期桶展开与返回父级全部改为无横向动画切换
- 优化日期桶惰性展开链路：更新后优先展开频道再展开日期桶，减少层级往返造成的抖动感
- 同步更新 `README.md`、`INSTRUCTIONS.md`、`UOSC_MODIFY_DIFF.md`

## V1.6.2 - 2026-03-23

- 日期桶升级为 7 桶动态显示：明天 / 今天 / 昨天 / 星期X（附月日副标题）
- 修复日期桶悬停联动：频道悬停可级联自动展开到当日 EPG，并定位当前时段到可视区域中间
- 优化日期桶交互：日期桶列改为不响应悬停选中，仅点击切换，且修复点击需两次才展开的问题
- 修复菜单宽度轻微联动：新增菜单级最大宽度约束，支持锁定三级日期桶宽度不随四级 EPG 波动
- 修复频道搜索崩溃：处理 `build_channel_menu_item` 多返回值导致的 `table.insert` 参数错误
- 修复拼音首字母模糊搜索不稳定：多音字映射改为稳定构建策略，`gd` 可匹配 `广东`，`dg` 可匹配 `东莞`

## V1.6.1 - 2026-03-23

- 优化三级 EPG 自动展开体验：鼠标悬停频道触发子菜单展开后，当前时段会自动选中并滚动到可视区域中间
- 优化回看场景三级 EPG 自动展开体验：观看回看时，菜单定位频道改为当前回看视频所属频道；鼠标悬停频道触发子菜单展开后，当前回看对应的 EPG 会自动选中并滚动到可视区域中间
- 菜单升级为四级结构：分组 > 频道 > 日期桶（明天/今天/昨天/前天/更早） > EPG
- 新增 `menu_level4_min_width` 配置项，用于第四级 EPG 菜单最小宽度
- 首次打开菜单默认构建到日期桶当天层（今天），提升直播频道场景的菜单响应速度
- 非预构建日期桶改为同树内惰性展开（不跳转菜单）

## V1.6 - 2026-03-23

- 支持在 `epg.conf` 分别设置一级/二级/三级菜单最小宽度：`menu_level1_min_width`、`menu_level2_min_width`、`menu_level3_min_width`
- 支持在 `epg.conf` 设置副标题字体大小：`menu_subtitle_font_size`
- 三级 EPG 菜单改为副标题模式：节目标题为主标题，日期时间改为副标题
- 三级 EPG 菜单项跟随极简样式，去掉额外图标/右侧提示

## V1.5 - 2026-03-23

- 缩短频道副标题，移除 `HH:MM-HH:MM` 时间段，减少二级菜单宽度
- 去掉频道右侧 `回看` / `EPG` 提示
- 去掉分组前面的 `folder` 图标
- 去掉所有子菜单默认 `>` 箭头
- 同步更新 `README.md`、`INSTRUCTIONS.md`、`UOSC_MODIFY_DIFF.md`
